## 👨‍💻 Sobre Mim

Desenvolvedor Java com experiência em construção de APIs RESTful utilizando Spring Boot, além de conhecimento sólido em bancos de dados como PostgreSQL e MongoDB. Estou atualmente me especializando em Engenharia de Dados pela CESAR School, ampliando meus horizontes com ferramentas modernas como Docker, Kubernetes e AWS.

Tenho um perfil colaborativo, gosto de resolver problemas complexos e estou sempre aprendendo. Busco oportunidades em ambientes inovadores, especialmente com foco em back-end e dados.
